Secure Installation

DentaCare Pro uses a hardened, CLI-based deployment process to ensure maximum security. Browser-based installers have been disabled.

Deployment Steps:

  1. Configure your web server (Apache/Nginx) with security rules.
  2. Set up environment variables using .env.example.
  3. Import the database schema via CLI.
  4. Rotate the generated admin credentials on first login.
