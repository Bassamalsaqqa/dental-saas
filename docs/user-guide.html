First Login & Security

Mandatory Setup

Upon your first login with the generated admin credentials, you must:

  1. Navigate to Profile Settings.
  2. Change your password immediately to a strong, unique value.
  3. Verify your email address for recovery.

Interactive Odontogram

Using the Odontogram

The interactive odontogram is a visual representation of all 32 teeth, allowing you to track conditions and treatments for each tooth.

Tooth Conditions

Healthy
Cavity
Filling
Crown
Root Canal

How to Use

  • Click on any tooth to edit its condition
  • Add treatment notes and dates
  • Changes are saved automatically
  • Print or export odontogram

Tooth Numbering System

The system uses the Universal Numbering System for tooth identification:

Upper Right

Teeth 1-8

Upper Left

Teeth 9-16

Lower Left

Teeth 17-24

Lower Right

Teeth 25-32
