# Verification: P0-06 Repo Secret + Dump Hygiene

## 1. Untracked Secrets & Dumps

Verify that `.env` and `.sql` files are not in the git index.

### Command:
```bash
git ls-files .env democa_dental.sql "*.sql"
```

### Expected Output:
(Empty output)

## 2. Ignore Verification

Verify that these patterns are in `.gitignore`.

### Command:
```bash
grep -E "^\.env$|^democa_dental\.sql$|^\*\.sql$" .gitignore
```

### Expected Output:
```text
.env
*.sql
```

## 3. No Staged Secrets

Verify that no hardcoded passwords or keys are being committed.

### Command:
```bash
rg -n "database.default.password|encryption.key" .
```

### Expected Output:
Only references to environment variables or documentation/placeholders. No actual plain-text secrets.
