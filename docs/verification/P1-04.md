# Verification: P1-04 Debug Exposure + /scripts HTTP Block

## 1. /scripts HTTP Block

Verify that direct access to scripts via HTTP returns 403 Forbidden.

### Command:
```bash
curl -I http://localhost/scripts/init_database.php
curl -I http://localhost/scripts/simple_login_test.php
```

### Expected Output:
```text
HTTP/1.1 403 Forbidden
```
*Note: This must hold true even if the web server document root is misconfigured to the project root.*

## 2. CLI-Only Guard

Verify that scripts can still be run via CLI.

### Command:
```bash
php scripts/simple_login_test.php
```

### Expected Output:
HTML output showing "Simple Login Test" (since it's being run from CLI, it passes the guard).

## 3. DBDebug Control

Verify `DBDebug` state in configuration.

### Command:
```bash
grep "DBDebug" app/Config/Database.php
```

### Expected Output:
```php
'DBDebug'      => (ENVIRONMENT === 'development'),
```
